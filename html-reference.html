What is HTML?

HTML is the standard markup language for creating websites and is usually used in tandem with other languages used for styling the website.

HTML Element

Uses

It is the top level element of an HTML page


Syntax

start tag:

<html>

end tag:

</html>

Example:

<html>
head and body
</html>
